[#_Promise__T_]
=== Promise< T >

*Package*: `TypeDB.Driver.Common`



A ``Promise`` represents an asynchronous network operation.

The request it represents is performed immediately. The response is only retrieved once the ``Promise`` is ``Resolve``d.
