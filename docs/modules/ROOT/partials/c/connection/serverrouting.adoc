==== Struct ServerRouting

`rust struct (pointer)`



``ServerRouting`` is used to represent server routing directives in FFI. It combines ``ServerRoutingTag`` and optional fields to form an instance of the original enum. ``address`` is not null only when tag is ``Direct``.
