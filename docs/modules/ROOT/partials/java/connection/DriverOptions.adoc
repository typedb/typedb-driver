[#_DriverOptions_serverDiscoveryAttempts_]
==== method ``serverDiscoveryAttempts``

[source,java]
----
@CheckReturnValue
public java.util.Optional<java.lang.Integer> serverDiscoveryAttempts()
----

Returns the value set for the server discovery attempts limit in this ``DriverOptions`` object. Limits the number of driver attempts to discover a single working server to perform an operation in case of a server unavailability. Every server is tested once, which means that at most: - {limit} operations are performed if the limit &lt;= the number of servers. - {number of servers} operations are performed if the limit &gt; the number of servers. - {number of servers} operations are performed if the limit is None. Affects every eventually consistent operation, including redirect failover, when the new primary server is unknown. 


[caption=""]
.Returns
`public¬†java.util.Optional<java.lang.Integer>`

[caption=""]
.Code examples
[source,java]
----
options.serverDiscoveryAttempts();
----
