=== PipelineStage.DeleteStage

`class`

*Package*: `com.typedb.driver.api.analyze`

*Superinterfaces:*

* `PipelineStage`

Represents a "delete" stage: 
[source,java]
----
 delete
     &lt;block&gt;;
     &lt;deleted_variables&gt;;
 
----
 e.g. 
[source,java]
----
 delete
     has $attribute of $owner; links ($player) of $relation;
     $deleted-instance;
 
----


[caption=""]
.Examples
[source,java]
----
delete
     <block>;
     <deleted_variables>;
----

, [source,java]
----
delete
     has $attribute of $owner; links ($player) of $relation;
     $deleted-instance;
----
