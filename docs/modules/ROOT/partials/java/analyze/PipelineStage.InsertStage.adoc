=== PipelineStage.InsertStage

`class`

*Package*: `com.typedb.driver.api.analyze`

*Superinterfaces:*

* `PipelineStage`

Represents an "insert" stage: insert &lt;block&gt; e.g. ``insert $f isa friendship, links (friend: $x, friend: $y)``
