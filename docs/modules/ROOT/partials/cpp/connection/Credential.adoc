[#_Credential]
=== Credential

*Package*: `TypeDB`



User credentials and TLS encryption settings for connecting to TypeDB Cloud.


[caption=""]
.Examples
[source,cpp]
----
// Creates a credentials with username & password over a plain-text connection.
Credential credentials(username, password, false);

// Creates a credentials as above, but the connection will be made over TLS.
Credential credentials(username, password, true);

// Creates a credentials as above, but TLS will use the specified CA to authenticate server certificates.
Credential credentials(username, password, "path/to/ca-certificate.pem");
----
